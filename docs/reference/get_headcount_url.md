# Get the download URL for a specific year

Constructs the download URL for MI School Data headcount files. URL
patterns change over time based on how CEPI has organized their files.

## Usage

``` r
get_headcount_url(end_year)
```

## Arguments

- end_year:

  School year end (e.g., 2024 for 2023-24)

## Value

Character string with download URL
